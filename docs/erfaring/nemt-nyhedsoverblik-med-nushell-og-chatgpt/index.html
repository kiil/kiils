Information overload - lad kunstig intelligens hjælpe dig

Af Lennart kiil
17/04/2023 - 5 minutter - 996 ord

Der er meget information og mange nyheder at forholde sig til.

Du kan bruge ChatGPT til at skabe bedre overblik.

Den metode som jeg vil vise her, består af to trin - første trin er at indhente de relevante data.

Der bruger jeg nushell og følgende nushell plugin:

https://github.com/nushell/nushell/tree/main/crates/nu_plugin_query

På den måde giver:

http get https://www.dr.dk/nyheder | query web -q 'h2 .dre-title-text '

Det er i sig selv ret kortfattet og giver et hurtigt overblik. Men ChatGPT kan fortætte det yderligere.

Jeg bruger så ChatGPT gennem følgende bash-script:

https://github.com/nitefood/ai-bash-gpt

Ved at benytte det script kan jeg lave “pipelines” - det vil sige jeg kan sende output fra den førnævnte kommando videre som input til ChatGPT som nu har forkortelsen ai. Sådan her:

http get https://www.dr.dk/nyheder | query web -q 'h2 .dre-title-text ' | ai 'giv et kort referat'

Og ChatGPT returnerer

  stof fundet i dansk drikkevand, tennisspiller Holger Rune er nu blandt de bedste, Røde Kors' genbrugsbutikker er en millionforretning, lunt vejr i hele landet, jazzmusiker Ahmad Jamal er død, kampe fortsætter i Sudan, Kinas forsvarsminister taler om bånd til Rusland, savnet kvinde i           
  Nordjylland er fundet, Odense Håndbold vinder over Aarhus, Aalborg Pirates slår Herning i ishockey, Brøndby stopper

  Superligaen, Pogacar vinder Amstel Gold Race, Dansk Folkepartis hovedbestyrelse får ny næstformand.

Da der i forvejen er tale om overskrifter, så er det ikke meget vundet ved den øvelse.

Men hvad kan ChatGPT så bruges til når dit input er i overskriftsform?

Med ai -c kan jeg bede ChatGPT om at fortsætte med at forholde sig til det første input.

Så giver jeg et nyt prompt

sorter efter sag
