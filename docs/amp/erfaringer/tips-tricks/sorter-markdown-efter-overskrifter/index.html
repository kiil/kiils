Sorter markdown dokument

Af Lennart kiil
27/07/2019 - Et minut - 62 ord

Sorter sektioner i markdown. For eksempel efter #

Sorter # b mellemrubrik efter # a mellemrubrik.

#B
a content of B

#A
b content of A

til

#A
b content of A

#B
a content of B

I terminal med

perl -0777 -ne '
    (undef,@paragraphs) = split /^#(?=[^#])/m;
    print map {"#$_"} sort @paragraphs;
' file.md
